# 🚀 Complex Form Validation & Dynamic DOM Manipulation

## 📌 Project Overview
This project is part of the Cognifyz Full Stack Development Internship (Level 2 – Intermediate).

The main goal of this task is to enhance form validation by implementing more complex rules and dynamically updating the webpage using JavaScript (DOM manipulation).

This project demonstrates how to create interactive and user-friendly web applications with real-time feedback.

## 🎯 Objective
- Implement advanced form validation
- Add dynamic updates to the DOM
- Improve user interaction and experience

## ✨ Features
- ✅ Advanced Form Validation (e.g., password strength, input rules)
- ⚡ Real-time feedback using JavaScript
- 🔄 Dynamic DOM Manipulation
- 🎯 Interactive UI updates based on user input
- 📱 Responsive and user-friendly design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
